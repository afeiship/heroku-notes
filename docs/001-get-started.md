# get started
> 快速开始。

## installation
```shell
# 1.1 install
brew tap heroku/brew && brew install heroku

# 1.2 install with nodejs
npm i -g heroku

# 2. check
heroku --version
# heroku/7.42.1 darwin-x64 node-v12.16.2
```
